# bike-purchase-analysis
Bike Purchase Behaviour Analysis
Objective

Analyze customer demographic data to identify factors influencing bike purchases.

Dataset

The dataset contains 1000 customer records including:

Age

Income

Commute distance

Profession

Car ownership

Home ownership

Tools Used

Microsoft Excel

Pivot Tables

Data Cleaning

Data Visualization

Key Insights

Middle-aged customers show the highest bike purchase rates.

Customers with shorter commuting distances are more likely to purchase bikes.

Individuals with fewer cars demonstrate stronger bike adoption.

Professionals show slightly higher purchase rates than other occupations.
